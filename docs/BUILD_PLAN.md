# Bringora Build Plan

## Phase 1

Build the private beta web app.

- Login gate
- Daily problem cards
- Text input
- DeepSeek API backend
- Structured output
- Copy button
- Usage limits

## Phase 2

Build simple organized notes.

- Organize notes card
- Category output
- Manual save later

## Phase 3

Prepare public launch.

- User accounts
- Usage allowance
- Saved outputs
- Support page
- Privacy page
- Terms page

## Rule

Bringora produces text strategy only. It does not generate video, image, or audio files.
