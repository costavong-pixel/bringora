# Bringora Product Spec

## Definition

Bringora is an AI daily thinking companion.

It converts messy human thoughts into one structured, useful text output.

It is not a prompt optimizer product. Prompt cleanup is only an internal engine to reduce wasted AI calls and improve response quality.

## Target Users

- Housewives and home managers
- Students
- Solo entrepreneurs
- Non-native English users
- People who know what they want but cannot organize or explain it clearly

## Brand Meaning

Ora is always with me.

Bringora should feel like a calm daily companion that helps users organize thoughts, make decisions, and take action.

## Core UX Principle

One input -> one chosen problem -> one structured output -> one next best action.

Avoid overwhelming users with five different outputs at once.

## Main User Problems

### Housewives / Home Managers

- Budget grocery planning
- Recipe ideas from available ingredients
- Family needs list
- Kids school supplies
- Discount tracking
- Daily tasks and bills
- Writing polite messages

### Students

- Explain confusing topics
- Summarize notes
- Study plans
- Assignment planning
- Email teachers
- Organize school tasks

### Solo Entrepreneurs

- Ads and promotion strategy
- Social post structure
- Customer replies
- Product/service descriptions
- Offer clarity
- Decision making
- Daily task planning

## MVP Cards

1. Help Me Write or Reply
2. Help Me Understand
3. Help Me Decide
4. Make a Step-by-Step Plan
5. Help Me Promote or Sell
6. Turn My Idea Into Something
7. Organize My Notes

## Output Rules

Every response must be text only.

Every response must contain:

1. Organized Summary
2. Category Breakdown or Strategy
3. Action Steps
4. Next Best Action

Do not output video files, images, audio, or multiple platform variants unless the user explicitly chooses that single output type.

## AI Provider

MVP uses DeepSeek direct API only.

No OpenAI dependency in MVP.

## Product Positioning

Wrong positioning:

- prompt tool
- AI content generator
- ChatGPT clone
- Notion clone

Correct positioning:

- AI daily thinking companion
- messy thought organizer
- life and business action helper
- simple Notion-like notepad without setup complexity
