# AppSumo Plan

## Positioning

Bringora is a daily AI thinking companion for people who want clarity without learning prompt engineering.

## Main offer

Lifetime access to Bringora web app with a monthly usage allowance.

## Target buyers

- Solopreneurs
- Content creators
- Students
- Home managers
- Small business owners

## Deal rules

- No unlimited AI usage
- Clear monthly usage allowance
- Text-only output
- DeepSeek direct API
- Saved outputs later

## Listing promise

From messy thoughts to clear action.

## Product must be ready before listing

- Working login
- Code redemption
- Clear onboarding
- Usage limits
- Support email
- Privacy page
- Terms page
